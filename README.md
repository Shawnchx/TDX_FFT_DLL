# 通达信FFT傅里叶周期分析DLL插件

通达信DLL插件，对股价序列做FFT傅里叶分析，输出频域Top9分量的幅值/周期/相位/频率，并提供合成拟合曲线。

## 快速开始

### 部署

1. 将 `v5/bin/x64/Release/TDX_FFT_DLL_v5.dll` 复制到 `<通达信安装目录>\T0002\dlls\`
2. 通达信公式管理器 → DLL函数管理 → 添加绑定（建议绑定到2号）
3. 将 `v5/` 下的公式代码粘贴到新的技术指标或选股公式中

> 各版本区别：
> - `v5/` — 最新版，含选股和排序公式，相位定义在最新bar
> - `v4/` — 中间版本，功能同v5但无选股公式
> - 根目录 — 初始版本，绑定3号

### 公式一览

| 公式文件 | 类型 | 功能 |
|---------|------|------|
| `v5/FFT_CYCLE_V5.txt` | 副图指标 | 归一化余弦分量曲线（Top1~Top9，幅值归一化到[-1,+1]） |
| `v5/FFT_SINE_V5.txt` | 副图指标 | 真实幅值余弦分量曲线（保留价格单位） |
| `v5/FFT_FIT_V5.txt` | 副图指标 | 合成拟合曲线 vs 收盘价 |
| `v5/FFT_SELECT_V5.txt` | 条件选股 | 多周期共振选股（A/B/C/D四策略） |
| `v5/FFT_RANK_V5.txt` | 条件选股 | 共振得分排序（按幅值加权向上动能打分） |

## 算法说明

### FFT参数
- **FFT_N = 8192**（最大点数，2^13），实际 N_USE 取 `MIN(dataLen, 8192)` 向下取2的幂
- **最小值**：64根K线
- **去线性趋势**：最小二乘 `y = a*t + b`，FFT对残差序列分析
- **幅值归一化**：`A = |X[k]| * 2 / N`（双边频谱转单边）
- **TOP_K = 9**：保留幅值最大的9个频率分量

### 重建公式
```
component(t) = A * cos(ω*t + φ_right)
fit(t) = trend(t) + Σᵢ Aᵢ * cos(ωᵢ*t + φᵢ)   (i=1..9)
```

### v5相位定义（与根版本/v4的区别）
- **根版本/v4**: φ 对应有效段起点（最早K线）的余弦相位
- **v5**: φ 对应最新K线的余弦相位，归一化到 [-π, π]
  - φ < 0 → 上升段，φ > 0 → 下降段
  - φ ≈ -π → 谷底附近（最强买点）
  - 可跨股票横向比较相位超前/滞后

### 函数编号速查（v5版）

| 编号 | 功能 | 输出类型 |
|------|------|---------|
| 1 | 触发FFT计算，直通Close | 序列 |
| 2 | 纯趋势线（线性拟合） | 序列 |
| 3 | 趋势+Top1 | 序列 |
| 4 | 趋势+Top1~2 | 序列 |
| 5 | 趋势+Top1~3 | 序列 |
| 6 | 趋势+Top1~4 | 序列 |
| 7 | 趋势+Top1~5 | 序列 |
| 8 | 趋势+Top1~6 | 序列 |
| 9 | 趋势+Top1~7 | 序列 |
| 10 | 趋势+Top1~9全合成拟合曲线 | 序列 |
| x1 | TopN 幅值（价格元） | 常量 |
| x2 | TopN 周期（bar数） | 常量 |
| x3 | TopN 相位（弧度，最新bar余弦相位） | 常量 |
| x4 | TopN 频率（cycle/bar） | 常量 |
| x5 | TopN 余弦分量（真实幅值） | 序列 |
| x6 | TopN 归一化余弦（幅值=1） | 序列 |

其中 x = 1~9，例如 Top1: 11~16, Top2: 21~26, ..., Top9: 91~96

| 项 | 精度 |

| FFT缓冲区 | complex<double> |
| 旋转因子表 | double |

## 编译说明

Visual Studio 2019/2022，工作负载"使用C++的桌面开发"。

```
msbuild TDX_FFT_DLL_v5.vcxproj /p:Configuration=Release /p:Platform=x64
```

## 注意事项

- DLL位数需与通达信匹配（新版x64，老版x86）
- 选股公式依赖 v5 相位语义（最新bar参考点），不可与根版本混用
